# Fotos de personas — Solvo Global

Retratos para photo cards de talento en piezas de marketing (landings, decks).

**Convención**: `person-01.jpg` … `person-08.jpg` (mínimo 8). Cuadradas, ~400×400px, rostro centrado — se muestran recortadas en círculo. Formato `.jpg` (los consumidores referencian esa extensión).

Fuente canónica de las fotos: `<vault>/shared/design-system/solvo-global/assets/people/`. Esta carpeta es una copia local commiteada en el bundle; al actualizar las fotos, re-copiarlas aquí (y al otro bundle que las use: `public/prototypes/email-cold-outreach-booking-landing-mvp/assets/people/`).

Mientras falte una foto, las cards muestran un retrato difuminado en CSS como fallback (no se rompe el diseño).
